The Difference between Margin and Padding

In this short blog post, we're going to see the difference between CSS Margin and CSS Padding using images. I'm going to use art exhibitions as an example to show the difference.

Art Display

Art paintings, in any exhibition, are placed on walls and each painting is separated with some space as in the photo below.

The photo above shows two paintings. There are a few things to notice in this photo. The paintings are framed and there is a visible whitespace between the edge of the paintings and the edge of the wooden frame. There is a small space between the frames. The painting on the right has more whitespace than the two other paintings.

Margin, Padding and Border

The space between the painting and the edge of the frame (marked in green) is called padding in CSS.
The space separating both frames (marked in red) is called Margin. Do you see the difference now? The wooden frame in CSS is the equivalent of Border.

Here is how you set these three properties in CSS. 

                

                .class-for-any-content {
                    margin: 10px;
                    padding: 10px;
                    border-color: black;
                    border-width: 2px;
                }
